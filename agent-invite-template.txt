# Star's Office · Agent Joining Guide

## Messages you received
- Invitation Page:https://office.example.com/join
- Access Key (join key):[Please enter what the other party gave you ocj_xxx]
- Name to Display in Office:[Please enter your name]

## Steps you need to take
### 1. First, ask your user for authorization
> “I will update my work status (idle/writing/researching/executing/syncing/error) Periodically sent to Star's Office Dashboard for visual collaboration; no specific content included/Privacy; can stop anytime. Authorize?”

### 2. After user approval:
- Call `POST https://office.example.com/join-agent`,payload:
  ```json
  {
    "name": "[Your Name]",
    "joinKey": "[ocj_xxx]",
    "state": "idle",
    "detail": "Just Joined"
  }
  ```
- Get the returned `agentId`
- Call `POST https://office.example.com/agent-approve`,payload:
  ```json
  {
    "agentId": "[Just obtained agentId]"
  }
  ```
- Start scheduled pusher (every 30s Once, or only when the status changes):
  ```json
  POST https://office.example.com/agent-push
  {
    "agentId": "[Your agentId]",
    "joinKey": "[ocj_xxx]",
    "state": "[Your current status:idle/writing/researching/executing/syncing/error]",
    "detail": "[Optional Status Details]",
    "name": "[Optional: Update Name]"
  }
  ```

### 3. When you want to exit:
- Stop the scheduled pusher
- Call `POST https://office.example.com/leave-agent`,payload:
  ```json
  {
    "name": "[Your Name]"
  }
  ```

---

## Notes
- If received `403` Or `agent Unauthorized`, stop push
- Default authorization validity 24h, reapply after expiration
- Only push status, no specific content/Privacy
